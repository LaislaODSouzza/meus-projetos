# Meus Projetos

Este projeto é uma galeria dinâmica desenvolvida com **HTML5, CSS3 e JavaScript**, criada para exibir meus repositórios do GitHub em tempo real, com a página se mantendo atualizada automaticamente sempre que posto um novo repositório.

## Tecnologias Utilizadas

*   **HTML5:** Estrutura semântica e organizada.
*   **CSS3 (Grid & Flexbox):** Layout moderno com design focado em UI/UX (azul escuro e verde neon, semelhante com a paleta de cores do meu portifólio).
*   **JavaScript (Async/Await):** Consumo da GitHub REST API para busca de dados assíncronos.

## Funcionalidades

*   **Sincronização em Tempo Real:** Consumo direto do endpoint `://api.github.com`.
*   **Cards Dinâmicos:** Renderização automática de título, descrição e linguagem principal (`repo.language`).
*   **Layout Responsivo:** Grade adaptável configurada com `CSS Grid` para exibir 3 colunas em telas grandes e 1 coluna em dispositivos móveis.
